## Overview
This repository contains a comprehensive analysis of Spotify's music dataset. The dataset includes detailed information about songs, artists, genres, release years, streams, and audio features such as danceability, energy, tempo, and duration. The goal is to explore music trends, identify top-performing tracks and artists, and generate insights into song characteristics and listening patterns.

## Objectives
- Identify the **top-streamed songs and artists**.  
- Analyze **song features** like danceability, energy, and tempo to understand trends in popular music.  
- Explore **song releases over the years** and detect patterns in music production.  
- Detect **duplicates, missing data, and anomalies** to ensure data quality.  
- Use **SQL queries** for filtering, aggregation, grouping, subqueries, joins, and window functions.  
- Optional: Visualize insights using Python or R.

- **Filtering & Sorting:** `WHERE`, `ORDER BY`, `TOP`
